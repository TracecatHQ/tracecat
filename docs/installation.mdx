Install Tracecat using Docker Compose.

## Prerequisites
- [Docker](https://docs.docker.com/get-docker/)
- [Docker compose](https://docs.docker.com/compose/install/)
- [Supabase account](https://app.supabase.io/)

## Clone GitHub repo

```bash
git clone git@github.com:TracecatHQ/tracecat.git
```

## Create `.env` file

```bash
TRACECAT__SIGNING_SECRET=${ openssl rand -hex 32 }
TRACECAT__SERVICE_KEY=${ openssl rand -hex 32 }
TRACECAT__DB_ENCRYPTION_KEY=your-fernet-key
SUPABASE_JWT_SECRET=your-supabase-jwt-secret
SUPABASE_JWT_ALGORITHM="HS256"
SUPABASE_PSQL_UR=your-supabase-psql-url
OPENAI_API_KEY=your-openai-api-key
```

*Note: to generate a `TRACECAT__SERVICE_KEY` you'll need to create a Fernet key: https://cryptography.io/en/latest/fernet/.*

## Start Tracecat

```bash
docker-compose up
```

You can access Tracecat's frontend at [http://localhost:3000](http://localhost:3000)
and the OpenAPI docs at [http://localhost:8000](http://localhost:8000).
