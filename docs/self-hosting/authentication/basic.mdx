<Warning>
  Basic auth should only be used for local development.
  For production, use [Google OAuth](/self-hosting/authentication/google-oauth) or [SAML SSO](/self-hosting/authentication/saml-sso).
</Warning>

# Configuration

In your `.env` file, make sure you have the following environment variables set:

```bash
TRACECAT__AUTH_TYPES=basic
```

## Sign-up

Create a new user to sign-in with a minimum password length of 12 characters.
