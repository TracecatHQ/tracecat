## `core.ssh.execute_command`

Execute a command over SSH and return stdout, stderr, and exit status.

### Inputs

<ResponseField name="command" type="string">

Required.

Command to execute on the remote host.

</ResponseField>

<ResponseField name="host" type="string">

Required.

SSH host to connect to.

</ResponseField>

<ResponseField name="host_public_key" type="string">

Required.

Expected public host key for the target host in '&lt;key_type> &lt;base64>' format. For non-22 ports, the entry is stored as [host]:port.

</ResponseField>

<ResponseField name="username" type="string">

Required.

SSH username.

</ResponseField>

<ResponseField name="password" type="string | null">

Optional.

Password for the SSH user.

Default: `null`.

</ResponseField>

<ResponseField name="port" type="integer">

Default: `22`.

</ResponseField>

<ResponseField name="timeout_seconds" type="number">

Optional.

Timeout in seconds for connection and command execution.

Default: `30.0`.

</ResponseField>
