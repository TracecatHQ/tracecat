Cases give your automations a place to store investigation state, analyst notes,
attachments, and linked records.

Use cases when you need to:

- Open and update work items from workflows
- Add comments, tags, attachments, and linked table rows during an investigation
- Route work between analysts and automated steps

## Work with cases in workflows

Use the [Cases core actions](/automations/core-actions/cases) page for the full
action reference, including:

- Creating, updating, searching, and deleting cases
- Adding comments, tags, attachments, and linked rows
- Managing case tasks

## Trigger workflows from case activity

Tracecat supports workflow triggers for case activity:

- [Case triggers](/automations/triggers/cases)
- [Task triggers](/automations/triggers/tasks)
- [Comment triggers](/automations/triggers/comments)
