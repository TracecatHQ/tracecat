The Tracecat API is organized around REST.
The API has predictable resource-orientated URLs and returns JSON-encoded responses.

With the Tracecat API, you can list, get, create, update, and delete actions, workflows, event logs, and cases.

You can find Tracecat's public OpenAPI documentation at [https://api.tracecat-engine.com/docs](https://api.tracecat-engine.com/docs).
